# DataFlow
Data Insight Dashboard

A modern and interactive data analytics dashboard built using Streamlit, Pandas, and Matplotlib.
This application allows users to upload CSV datasets and instantly generate insights, summary statistics, and visualizations through a clean and aesthetic user interface.

---

## Features
1. Upload CSV datasets
2. Interactive dashboard layout
3. Automatic data insights
4. Histogram visualization
5. Missing value detection
6. Clean and modern UI design
7. Responsive wide layout

# Installation
1. Clone this repository:
```bash
git clone https://github.com/your-username/data-insight-dashboard.git
```
2. Move into the project folder:
```bash
cd data-insight-dashboard
```
3. Install dependencies:
```bash
pip install -r requirements.txt
```
4. Run the Streamlit application:
```bash
streamlit run data_ai_app.py
```

# How It Works
1. Upload a CSV file
2. The dashboard reads the dataset using Pandas
3. Key metrics are automatically generated
4. Insights are calculated for numeric columns
5. Histograms are visualized interactively
